---

## About This Workshop

A practical workshop covering the full pipeline for organizing, describing, and exposing image corpora for humanities research.

**Part 1 — Conceptual Foundations**
- Semantic Web, RDF triples, URIs
- Metadata schemas: Dublin Core, CIDOC-CRM
- Controlled vocabularies: GeoNames, Pactols, Openthéso

**Part 2 — IIIF**
- IIIF Image API and Presentation API (Manifests)
- IIIF in Europeana, Omeka S, and Nakala

**Part 3 — Tools**
- **Tropy** — organize and describe research photos
- **Arvest** — IIIF import, collaborative annotation, API exposure
